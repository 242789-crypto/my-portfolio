# Portfolio CMS

A modern, responsive portfolio website with a built-in Node.js CMS.

## Features
- **Dynamic Content**: Manage your Hero, About, Skills, and Portfolio sections via a private admin panel.
- **Responsive Design**: Optimized for both mobile and desktop devices.
- **Premium Aesthetic**: Dark theme with amber accents and smooth animations.

## Local Setup
1. Clone the repository (or download the files).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   node server.js
   ```
4. Access the Website: `http://localhost:3000`
5. Access the Admin Panel: `http://localhost:3000/admin.html`
   - **Username**: admin
   - **Password**: waleed123

## Deployment
For full CMS functionality, it is recommended to deploy on a platform that supports Node.js and persistent storage (like Render or Railway).
